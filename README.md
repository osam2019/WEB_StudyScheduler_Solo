##Study Scheduler

스터디를 계획, 모집, 그리고 관리할 수 있는 웹 서비스입니다. 

개발자/팀 : (공군) 일병 박상민 / S2

## Project setup
```
npm install
npm run serve
```

## JSON server setup
```
json-server ./src/assets/data.json
```

Then, web service may be available at localhost:8080
